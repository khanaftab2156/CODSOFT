# Movie Rating Prediction

## Overview

This project predicts movie ratings using Machine Learning techniques. The dataset was cleaned, preprocessed, and used to train predictive models.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Workflow

* Data Cleaning
* Handling Missing Values
* Removing Duplicates
* Feature Encoding
* Train-Test Split
* Model Training (Linear Regression & Random Forest)
* Model Evaluation

## Results

Random Forest performed better than Linear Regression with:

* MSE: 0.65
* R² Score: 0.32

## Conclusion

Random Forest was selected as the final model due to its better predictive performance on the movie rating dataset.
